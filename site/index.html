Proof language quick reference
theorem NAME [using LOGIC]: — theorem header
assume LABEL: FORMULA — introduce an assumption
suppose LABEL: FORMULA — subproof assumption (indented body)
have LABEL: FORMULA by RULE REFS… — derive a step
conclude FORMULA by LABEL — final conclusion

Rule Soundness

        

          Check whether each inference rule of a proof logic is sound
          under a semantic matrix.  This is a semantic diagnostic
          (⊨), separate from proof checking (⊢).
        

        

          
          
          
          
          
        

        

      


      
      

        
World Lattice (CH-Style Independence)

        

          Checks a formula's semantic status across three worlds (Γ, Γ+φ, Γ+¬φ)
          demonstrating proposition-level independence patterns.
          This is a toy analogue of forcing, not set-theoretic independence.
